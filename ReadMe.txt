Project Title: Trython.

Getting Started - In order to run this application on you mobile device, you will need to 
	          download the application from the online store of you device.


Prerequisites - The only requirements that is needed to run this application is an Android OS and the 
		program itself.


Installing - Step 1) Open you device's app store.
	     Step 2) Search for the application "Trython".
             Step 3) Run the application after download to device is complete.


Deployment - Deploying the live system only requires the user to run the application 
             from the device they are using. 


Built With - Android Studio
